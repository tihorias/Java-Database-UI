Welcome to the Bank Database

The program connects to the Ryerson Oracle Database and lets us create, populate, alter and drop tables.

How to run:
Either 1 or 2
1. If your VPN works, you can directly run the MainGUI.jar
2. If your VPN works, you can use Eclipse and build the library using ojdbc6.jar and rs2xml.jar and then compile MainGUI.java


Structure of the program: 

Create Table:
Creates a table called RESERVE3

Populate Table:
Inserts data into the RESERVE3 table. (reserve_id, reserve_amount, reserve_location)

Alter Table:
Lets a user delete a row by using the reserve_id as an argument.

Delete Table:
Drops the table RESERVE3
